环境准备

hdc 命令使用说明

查询相关的命令

-h/help 查看hdc帮助

-v/version 查看hdc版本信息

list targets [-v] 查看所有已经连接的设备列表

服务进程相关命令

hdc kill [-r] 终止hdc服务进程

target boot 重启目标设备

smode [off] 授予设备端hdc后台服务进程root权限， 使用off参数取消授权

target mount 读写模式挂载系统分区

APP相关命令

hdc install packageFile 安装指定的应用

hdc uninstall packageName 卸载指定包名的应用

文件相关命令

hdc file send local remote 从本地发送文件至远端设备

hdc file recv remote local 从远端设备发送文件至本地

调试相关命令

hdc shell [COMMAND] 交互命令，COMMAND表示需要执行的单次命令。其中当OS镜像API≥9时，不支持交互式进入设备，仅支持交互式命令。

官方链接

hdc使用指导

注意事项

  • 使用hdc，如果出现异常，可以尝试通过hdc kill命令杀掉hdc服务，或者通过hdc start -r命令重启服务进程进行解决。
  • 如果出现hdc list targets获取不到设备信息，通过任务管理器查看是否有hdc进程存在，如果进程存在，可以通过杀掉该进程进行解
  • 吐槽一下：mac 环境下不太好使！！！
