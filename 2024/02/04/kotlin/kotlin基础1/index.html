空安全性

安全调用操作符： 使用 ?. 操作符可以在调用可能为空的对象的方法或访问其属性时进行安全调用。如果对象为 null，则表达式返回 null，而不会导致空指针异常。

Elvis 操作符（Elvis Operator）： ?: 操作符用于在对象为 null 时提供默认值。

非空断言操作符（Non-null Assertion Operator）： !! 操作符用于将一个可空类型的对象转换为非空类型。如果对象为空，则会抛出空指针异常。

安全类型转换操作符（Safe Cast Operator）： as? 操作符用于安全地将一个对象转换为指定的类型。如果转换失败，结果将为 null。

集合操作函数：

map()

对集合中的每个元素应用指定的函数，并返回结果集合。

zip()

将两个集合合并为一个具有相同元素数量的新集合。这个新集合的每个元素都是由原始两个集合中对应位置的元素组成的。如果两个集合的大小不同，zip 函数会以较小的集合的大小为准进行合并。

filter()

过滤集合中满足条件的元素，并返回满足条件的元素集合。返回true或false确定是否过滤

reduce()

使用指定的函数将集合中的元素累积成一个结果。

flatMap()

将集合中的每个元素映射为另一个集合，并将结果扁平化为单个集合。

groupBy()

根据指定的键函数将集合中的元素分组。

sortedBy()

根据指定的条件对集合进行排序。

maxBy() / minBy()

返回集合中根据指定条件最大 / 最小的元素。

其他常用函数：

let

在对象(任何对象)上执行操作，并返回结果。

apply

在对象上执行操作，并返回对象本身。apply函数始终是返回对象本身 ，持有this，表示当前对象

run

在一个上下文中执行操作，可以引用上下文对象而无需重复名称。

also

在对象上执行操作，并返回对象本身，与 apply 类似，但返回的是对象本身，函数内的操作不会影响对象本身。

takeIf / takeUnless

根据条件选择是否采取某种操作。

takeIf接受一个 lambda 表达式作为参数，该 lambda 表达式的参数类型为对象类型，并返回一个布尔值。如果 lambda 表达式返回 true，则 takeIf 函数返回对象本身；如果 lambda 表达式返回 false，则 takeIf 函数返回 null

takeUnless 则相反，如果 lambda 表达式返回 true，则返回 null；如果 lambda 表达式返回 false，则返回对象本身。

require: 断言条件是否满足，如果不满足，则抛出异常。
